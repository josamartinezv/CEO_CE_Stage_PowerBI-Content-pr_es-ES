# Preguntas más frecuentes acerca de Power BI

## ¿Qué es Microsoft Power BI?


            [Power BI](https://powerbi.microsoft.com/) es un servicio de análisis de negocio basadas en la nube que permite a cualquier persona visualizar y analizar datos con mayor velocidad, eficiencia y descripción. Los usuarios se conecta a una amplia gama de datos a través de fácil de usar paneles, informes interactivos y visualizaciones atractivas que aportan los datos para la vida.

## ¿Cuál es la diferencia entre Power BI y Power BI Pro?

Power BI proporciona todo tipo de características para ayudarle a empezar a explorar los datos de una forma totalmente nueva. Power BI Pro ofrece todas las mismas características excelentes en Power BI, además de características adicionales, como más capacidad de almacenamiento, datos de programación de actualización más frecuentes que los orígenes de datos en directo, diario con interactividad completa, grupos y mucho más. Obtenga más información sobre [las diferencias entre Power BI Pro y la versión gratuita de Power BI](https://powerbi.microsoft.com/pricing).

## ¿Cuánto cuesta Power BI?

Power BI y Power BI Desktop son gratuitas. Hay una versión de prueba gratuita de 60 días para Power BI Pro. Después de eso, Power BI Pro es $9.99/User/Month. Más información sobre los precios de nuestros [Power BI precios](https://powerbi.microsoft.com/pricing).

## ¿Qué es Power BI Desktop?


            [Power BI Desktop](https://powerbi.microsoft.com/designer) es una aplicación de escritorio gratuita que se puede instalar directamente en su propio equipo. Power BI Desktop funciona conjuntamente con el servicio Power BI proporcionando exploración avanzada de datos, modelado, modelado y creación de informes con visualizaciones altamente interactivas. Puede guardar su trabajo en un archivo y publicar los datos e informes de derecho en el sitio de Power BI para compartir con otros usuarios.

## ¿Qué es necesario usar Power BI?

Dirección de correo electrónico un explorador Web y trabajo.


            **Nota:** actualmente no se admiten direcciones de correo electrónico de trabajo que terminen en .gov y. mil.

## ¿Por qué es necesario iniciar sesión con mi correo electrónico del trabajo

Power BI no admite las direcciones de correo electrónico proporcionadas por los proveedores de telecomunicaciones o de servicios de correo electrónico de consumidor. Obtenga más información sobre [el proceso de suscripción del autoservicio de Power BI](powerbi-service-self-service-signup-for-power-bi.md).

## ¿Se admiten las direcciones de correo electrónico de trabajo?
Trabajo de direcciones de correo electrónico que terminen en **.edu** y **.org** son compatibles.

Las que terminan en **.gov** y **. mil** no se admiten actualmente.

## ¿Es el Power BI disponibles de forma local?

No, Power BI no está disponible como un servicio de nube privada, interna. Sin embargo, con Power BI y Power BI Desktop, segura puede conectarse a sus propios orígenes de datos locales. Con el [puerta de enlace de datos local](powerbi-gateway-onprem.md), puede conectarse a su servidor de SQL Server Analysis Services local y otros orígenes de datos en directo. También puede actualización programada con una puerta de enlace centralizada. Si una puerta de enlace no está disponible, puede actualizar los datos de orígenes de datos locales mediante el [Power BI Gateway - Personal](powerbi-personal-gateway.md).

También puede ver informes de dispositivos móviles de SQL Server local con las aplicaciones de iOS de Power BI:

-   
            [Informes de SQL Server mobile en el iPhone](powerbi-mobile-iphone-kpis-mobile-reports.md).
-   [Informes de SQL Server mobile en el iPad](powerbi-mobile-ipad-kpis-mobile-reports.md)

## ¿Qué orígenes de datos puedo conectarme?

La lista de orígenes de datos para Power BI es amplia, pero puede agruparse en lo siguiente:

- Datos de [archivos de Excel y Power BI Desktop](powerbi-service-get-data-from-files.md).

- 
            [Paquetes de servicios de contenido](powerbi-content-packs-services.md), con conjuntos de datos para servicios como Salesforce, informes y paneles listos para usar. Además de establecer una conexión de datos, Power BI proporciona informes y paneles pregenerados para cada uno de estos servicios.

- Conectores a otros conjuntos de datos como bases de datos [base de datos de SQL Azure](powerbi-azure-sql-database-with-direct-connect.md) y SQL Server [Analysis Services](powerbi-sql-server-analysis-services-tabular-data.md) datos tabulares.

Obtenga más información sobre [obtener datos](powerbi-service-get-data.md) en Power BI.

## ¿Qué son los paquetes de contenido?


            [Paquetes de servicios de contenido](powerbi-content-packs-services.md) previamente generadas soluciones para servicios populares como parte de la experiencia de Power BI. Conectarse a su cuenta de Power BI y ver sus datos a través de live paneles e informes interactivos que se han generado previamente para ellos rápidamente un suscriptor a un servicio compatible.  Se ha lanzado paquetes de contenido para servicios populares, como Salesforce.com, Marketo, análisis de Adobe, Azure Mobile Engagement, CircuitID, comScore Digital Analytix, Quickbooks Online, SQL Sentry y tyGraph. Durante los próximos meses, se amplía para incluir paquetes de contenido e integraciones de Sage, SpaceCurve, Sumo lógica, Zuora, eRoom Planview, Insightly, Troux, permitirán saber hasta y otras personas.


            [Paquetes de contenido organizativo](powerbi-service-organizational-content-packs-introduction.md) proporcionan a los usuarios, profesionales de BI e integradores de sistema, las herramientas para crear sus propios paquetes de contenido para compartir especialmente paneles, informes y conjuntos de datos dentro de su organización.
